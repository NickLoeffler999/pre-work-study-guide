CSS

  • Enter your CSS notes here
  • CSS defines the colors, aesthetics, visual layout, and organization of our project.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • .card is used as a selector and needs to be included as a "class" before each id to have said style applied.

Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it.
  • in Git Bash, the cd command takes us to the directory of our choice.
  • git add -A is used to commit all files in the current working branch
  • git commit -m is used to associate a message with our commit
